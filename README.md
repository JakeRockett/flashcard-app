# Flashcard App

A Kotlin console application for creating, managing, and studying flashcards, organized by subject. This project demonstrates CRUD operations, persistence using XML/JSON, and Test-Driven Development with JUnit 5.

---

## Features

- **Flashcard Management**
  - Add new flashcards
  - Update or delete existing flashcards
  - List all flashcards or by subject
- **Subject Management**
  - Organize flashcards by subjects
  - Add flashcards to specific subjects
  - Study flashcards per subject
- **Persistence**
  - Save and load flashcards in XML or JSON format
- **Testing**
  - Full unit tests covering add, update, delete, search, and persistence operations
